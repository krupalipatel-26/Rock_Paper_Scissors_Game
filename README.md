# Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built using Python. The player competes against the computer, which randomly selects its move. The game determines the winner based on the classic Rock-Paper-Scissors rules and keeps track of scores.

## Features
- Play against the computer
- Random computer choices
- Score tracking system
- Input validation
- Replay option
- User-friendly console interface

## Technologies Used
- Python
- Random Module

## Learning Outcomes
- Python Fundamentals
- Conditional Statements (if-elif-else)
- Loops (while)
- User Input Handling
- Random Number Generation
- Game Logic Development

## How to Run

```bash
python rock_paper_scissors.py
```
